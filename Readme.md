    [得意黑的故事](https://www.bilibili.com/video/BV1sP411g7PZ/?spm_id_from=333.999.0.0)

      这款字体是一款具有美术字风格的窄斜体中文字体，并且是基于 SIL Open Font License v1.1 开源协议发布的。它参考了80年代手绘美术字的字形特征，整体倾斜6度，字宽对比全身压窄20%，字形窄长，重心上提，字体整体昂扬着积极的气质和明快的速度感。

      西文部分为了保证混排时视觉一致性而将倾斜角度调整至8度。

      得意黑的汉字部分覆盖了简体中文字符集 GB-2132 中的6763个汉字，并且西文部分覆盖了泛欧、美洲、南亚各种语言所需的字符共415个，支持100余种语言 。

    注意事项

      - 本字体部分笔画和结构采用了美术字风格的形态处理方式，并未严格遵循中国大陆规范汉字写法，敬请注意。

      - 使用场景：

        - 推荐：平面海报、电商文案、视频标题及字幕、书报标题、游戏嵌入等；

        - 不推荐：正文排版、编程代码、手机界面等。

  - [方正屏显雅宋](https://www.foundertype.com/index.php/FontInfo/index/id/402.html)（商用需授权）：针对屏幕显示，**调整横竖笔画比例**，减少传统宋体在屏幕上横竖粗细差异过大的问题。



    方正屏显雅宋的核心价值在于其**针对性优化**。传统宋体（如标雅宋）为纸质印刷设计，其笔画粗细对比强烈，但在像素密度有限的屏幕上，过细的横画可能显得模糊或产生锯齿感，影响长时间阅读的舒适度 。屏显雅宋通过**加粗横画**，缩小了横竖笔画间的视觉差距，使得文字在屏幕上显示时**轮廓更清晰、整体感更强**，有效提升了可读性 。同时，它保留了宋体字本身的**书法骨架和人文气息**，避免了为追求清晰而完全牺牲传统美感的遗憾 。
